## 测试流程（按顺序执行，全部走通才算完成）

1. **正向流程**
   - 启动前后端
   - 点击「添加实体」，添加 `User`、`Post`、`Comment` 三个实体
   - 为每个实体添加字段（至少 3 个字段，含 PK）
   - 连线：User → Post（一对多），Post → Comment（一对多）
   - 分别生成 SQLAlchemy / Django / SQL 代码，验证语法正确
   - 导出 JSON、PNG，验证文件内容

2. **保存/加载**
   - 保存项目
   - 刷新页面
   - 加载项目，验证画布恢复（实体、字段、连线均在）

3. **反向解析**
   - 准备一个有外键关系的 MySQL 数据库（或用测试库）
   - 点击「连接数据库」，填写连接信息，点击「解析」
   - 验证画布自动显示表结构和关系
   - 对解析结果生成 SQLAlchemy 代码

4. **错误处理**
   - MySQL 填写错误密码，验证报错提示清晰
   - 空画布点击「生成代码」，验证无崩溃（返回空代码或提示）
